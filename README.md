# RAG-Frontend

## Prerequisites

- Node.js version 20.10.0

## Setup Instructions

### 1. Install Dependencies:

   ```bash
   npm i
   ```

### 2. Configure Environment:

   ```bash
   cp .env.example .env
   ```

   Add values for the environment keys in the .env file.

### 3. Run the Server:

   ```bash
   npm run dev
   ```

   On the localhost:5173 port, you will see the webpage:
